# EURECA+ 건설공사 원가계산 시스템 — 작업 내역

> 작성자: **김성욱**
> 대상 시스템: EURECA+ (건설공사 원가계산 및 단가자료 관리 시스템)
> 기술 스택: Spring Boot 2.7 · Java 17 · MyBatis · Oracle · Pebble · AUIGrid/SlickGrid

건설공사 단가자료(노무·자재·기계경비·일위대가 등)를 등록·관리하고 원가계산 결과를 산출하는 시스템의 백엔드·프런트엔드 기능을 담당했습니다. 아래는 담당·개발한 모듈과 주요 기능을 정리한 것입니다.

## 2. 주요 개발·개선 내역 (2026)

### 원가계산 핵심 로직
- **원가계산 제비율** 규모구간 정비 — 일반관리비·이윤·고용보험·간접노무비·기타경비의 공사규모별 요율 체계 추가·변경
- **일위대가 손료 계산/재계산** — 비목별 분배 로직, 소수점 처리(0.1원 미만 버림 후 합산), C3R 파일 업로드 시 손료 자동 재계산 및 합계 갱신
- **전체계산 단가 오류 수정** — 표준시장단가 0원 산출 버그 등

### 원가계산 사용성 개선
- 자료 **찾아가기** 팝업 및 Context Menu(찾아가기·새로고침) 기능
- **표시순서 드래그** 방식 개편 + Context Menu 표시순서 변경
- 산정근거 **소계·총계** 버튼, 경비항목/일위대가 하위항목 **미리보기**
- 변경내역서 작성완료 처리 및 표시순서 정책 정비

### 물가지·자재 자동화
- **물가지-자재 자동매핑** — 매핑 스코어 정규화, 자동매칭 중단 기능, 저장 방식 이관(stateless)
- **물가지 정보 자동적용** — 인도조건 세그먼트 파생, 저장 키 정비, 기준월(물가적용년월) 기준 일원화
- **물가지 정보 검색** — 그룹핑 해제 조회, 원본가 표시, 유사도 기반 필터링, 소수 단가(유류 등) 표시, 검토 필요 안내
