# 건설 원가 산출 시스템 — 기술 포트폴리오

공공 물가정보와 시중단가를 수집·매핑하여 건설 원가를 산출하는 시스템류에서 다룬 백엔드/풀스택 엔지니어링 과제를, 특정 제품과 무관하게 **적용한 기법 중심으로 재구성**한 저장소입니다. 대용량 단가 데이터의 자동 매핑, 유사도 기반 검색, 조회 쿼리·API 성능 개선을 실무 관점에서 정리했습니다.

## 기술 스택

- **언어/런타임**: Java 17
- **프레임워크**: Spring Boot 2.7.x (Spring MVC, Spring Security)
- **데이터 접근**: MyBatis (동적 SQL, 매퍼 XML)
- **DB**: Oracle (PL/SQL, 계층 쿼리, 인덱스 튜닝)
- **뷰**: 서버사이드 템플릿 엔진 (SSR)
- **빌드**: Gradle

### 01. 외부 물가정보 자동적용 · 유사도 기반 정보검색
외부에서 수집한 가격정보(품목·규격·단가)를 내부 원가 항목에 조인 키 기반으로 자동 매핑하고, 정규화된 유사도 점수와 신뢰도 등급으로 후보를 정렬하여 검색·적용하는 로직을 다룹니다. "채움(가격 슬롯 채우기)"과 "확정(적용 플래그)"을 분리한 상태 관리 정책을 포함합니다.

### 02. 대용량 단가 조회 쿼리 / API 성능개선
조인 키 파생(부분 문자열) 기반 인덱스 도입으로 풀 테이블 스캔을 제거하고, 스칼라 서브쿼리 제거·배치 처리 메모리 개선·계층 쿼리 정합성 확보 등으로 조회 및 집계 API의 지연을 줄인 과정을 정리합니다.
